# Architecture Decision Records (ADRs)

> Index of all major technical decisions made during the FraudGuard project.
> Each ADR is immutable once Accepted. Superseded decisions remain visible for context.

## What is an ADR?

A short document capturing one technical decision: the context that forced it, what was chosen, what alternatives were considered, and what the tradeoffs are.

Standard fields: Status, Date, Context, Decision, Alternatives Considered, Consequences.
